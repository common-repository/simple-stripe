== Description ==

Accept credit card payments via Stripe payment gateway.
This plugin is suitable for very small payments without having to build an e-commerce site.


== Installation ==

1. Upload `simple-stripe` folder into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
